# yt-dlp Ez GUI
A simple web application for downloading videos using **yt-dlp**.  
Built with **Python** to provide an easy-to-use web interface for downloading videos from supported platforms.
- For who dont know how to use it in CMD
## Features

- 🎥 Download videos from supported websites using yt-dlp
- 🌐 Simple web interface
- ⚡ Fast and lightweight
- 🐍 Built with Python

## Requirements

- Python 3.10+
- yt-dlp
- FFmpeg (recommended)
- Dependencies listed in `requirements.txt`

## Installation

1. Clone the repository

```bash
git clone https://github.com/Caramel0784/ytdlp-web-test.git
cd ytdlp-web-test
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. (Optional) Install FFmpeg and make sure it is available in your system PATH.

## Run

Start the application with:

```bash
python app.py
```

Then open your browser and visit the local address displayed in the terminal.
